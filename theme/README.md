# Theme Workspace

This directory is reserved for the Shopify theme.

Recommended next options:

1. Pull an existing Shopify theme with `shopify theme pull --store <store>.myshopify.com`.
2. Initialize a new Dawn-based theme here for preview work.
3. Convert this directory to a git submodule after creating a paired theme repository, for example `HuaqingAI/outdoor-gear-shop-theme`.

Do not commit Shopify CLI session files or store credentials.
